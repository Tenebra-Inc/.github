# Tenebra Incorporated

## 1. Who are we?

Tenebra Inc. is a collective engaged in software development and interactive media production.
The collective operates as a GitHub organization for version control, issue tracking, and distribution of source code.

The same name and visual identity may appear as a fictional corporate entity within certain game products.

No legal or operational connection exists between these two uses beyond naming convention.

## 3. Interaction

Members are advised to:

<!-- it's a good practice, unsigned commits may be denied later -->

- Configure GPG signing for commits.
- Use ProtonMail or equivalent encrypted email as their GitHub account primary email.
- Enable two-factor authentication on their GitHub account.

Public repositories may be forked or cloned without restriction. Pull requests from non-members are accepted, but reviewed under the same criteria as internal contributions. Commentary unrelated to code or documentation may be ignored by the respective project maintainer.

## 4. Communication

Official Organization bulletins are distributed as:

- Audio and video materials
- Plain text transcripts

Transcripts should be signed and/or encrypted by the Organization GPG key. Unsigned transcripts are not to be trusted or linked to the Organization. Bulletin issue frequency is irregular.

For technical matters related to a specific repository, use that repository's issue tracker. For security reports, follow its security policy.

Public GPG keys are available in the `.github` repository of our Organization.

Unrelated, general inquiries are accepted with low priority.
